ThaiKitchen – Food Delivery App

Hi there! This is my web project for a food delivery app, heavily inspired by the layout and functionality of popular delivery platforms like Lieferando or UberEats. 

The project was built based on a specific set of design and functional requirements, covering everything from dynamic data rendering to a fully responsive layout.

Key Features

* Dynamic Menu 
* Interactive Basket
* Smooth Checkout Flow
* Fully Responsive


Code Architecture & Best Practices

Keeping the codebase clean, readable, and highly structured was a top priority for me:

* Keep it Clean
* Template Separation
* The 14-Line Rule
* Safe Initialization


Project Structure
├── assets/
│   ├── icons/          # Icons für Sterne, Warenkorb, Buttons & Favicon
│   └── img/            # Bilder der Gerichte und Hero-Banner
├── scripts/

├── script.js           # Haupt-Logik (Warenkorb & Modal-Steuerung)
├── style.css           # Haupt-Stylesheets (Desktop)
└── README.md           # Projektdokumentation
